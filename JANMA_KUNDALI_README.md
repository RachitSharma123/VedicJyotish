# Janma Kundali Tab (New File)

This adds a **new Streamlit file** without modifying existing app files:

- `streamlit_janma_tab.py`

## Run

```bash
streamlit run streamlit.py
# then choose "Janma Kundali" module
```

## What it includes

- Separate tabs:
  - `Prashna Reading`
  - `Janma Kundali (Birth Chart)`
- Sidereal-style planetary longitudes (Navagraha) with Lahiri ayanamsa approximation.
- Nakshatra, Panchanga (Tithi, Yoga, Karana), and Vimshottari Dasha starter calculations.
- QUESTION WRITING AREA and AI-enhanced Vedic guidance.
- Conclusion block for final actionable summary.

## Notes

This is a **starter birth-chart module**. It is designed to be extended with higher-precision ephemeris calculations and deeper divisional-chart logic in follow-up iterations.
